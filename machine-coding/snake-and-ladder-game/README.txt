Snake Ladder Game

Features -
1. Multiple players can play the game
2. Players roll a 6-sided dice
3. Board of 100 cells, numbered from 1-100
3. Ladder takes players ahead in the game from point A to point B (bottom to top)
4. Snake takes players behind in the game from point B, back to point A (top to bottom)


Entities -
1. Player
2. Snake
3. Ladder
4. Board


Services -
1. snake_ladder
2. dice
